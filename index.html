Implementations

  • Face Detection:
    use Cascade Classifier and Haarcascade_frontalface "Rainer Lienhart and Jochen Maydt. An Extended Set of Haar-like Features for Rapid Object Detection. IEEE ICIP 2002, Vol. 1, pp. 900-903, Sep. 2002" Implemented by OpenCV (http://docs.opencv.org/modules/objdetect/doc/cascade_classification.html?highlight=cascadeclassifier)

  • Tracking:
    use Camshift algorithm to track certain area (in our case, the biggest face) "Bradski, G.R. °ßComputer Vision Face Tracking for Use in a Perceptual User Interface°®, Intel, 1998" Implemented by OpenCV (http://docs.opencv.org/modules/video/doc/motion_analysis_and_object_tracking.html)

  • Control:
